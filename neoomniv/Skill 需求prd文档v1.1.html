Skill 市场 PRD 文档

版本 1.1 | 2026/06

一、产品概述

产品名称：技能市场平台

产品定位：企业级技能市场平台，支持企业自有技能市场及第三方技能市场浏览、安装

目标用户：企业内部用户

功能概述：

  1. 企业skill：支持技能的创建、导入、编辑、发布/下架、更新版本、批量操作及回收站管理。技能市场安装的技能不可编辑，自己创建或导入的技能支持编辑
  2. 第三方 Skill 市场无缝接入：固定获取技能清单中的技能预先加载到服务器，作为技能市场；支持手动从讯飞（https://skill.xfyun.cn/）官方技能库获取最新版本并预装到系统内，提供多维度的分类筛选及一键安装能力。
  3. 风险管控：内置严格的异步风险检测机制。在技能导入与新增环节，系统会自动扫描并拦截包含危险代码（如 eval/exec）等的高风险技能。

二、功能模块

2.1 企业 Skill 模块

界面功能

  • 模块标题：显示"企业 skill"，包含全部/市场/自定义切换按钮，默认显示全部
  • 搜索框：支持技能名称关键字搜索
  • 批量操作栏：位于模块右上角，包含全选复选框（支持一键全选当前显示的技能）、批量发布、批量下架、批量删除按钮；选中数量提示（如"已选3个"）
  • 技能卡片：显示圆角方形图标（默认首字母+随机底色）、技能名称、描述、来源标签（官方/自定义）、创建者、更新时间、版本号、状态开关（发布/下架）、使用次数
  • 展开/收起：默认展示第一行技能，支持展开查看更多；展开后如超出5行则右下角显示分页
  • 添加技能：右上角下拉显示创建技能、导入技能
  • 回收站入口：右上角显示回收站（数量）
  • 分页：每页8个，自适应显示，超出8个，底部显示分页

交互功能

  • 全部/市场/自定义切换：点击切换显示全部技能或显示来源市场/自定义的技能
  • 查看详情：点击技能卡片打开详情页（新页面，不离开当前页）
  • 发布/下架切换：点击卡片上的开关直接切换状态，无需二次确认
  • 删除技能：鼠标悬浮在卡片上显示删除按钮，点击弹出确认框"确认删除该技能吗？"，确认后从企业skill列表消失并移至回收站
  • 批量删除：弹出确认框"确定删除X个技能吗？"，确认后批量移至回收站
  • 展开更多：技能超过4个时显示「展开」按钮，点击加载更多；分页后支持收起
  • 编辑技能：所有技能，鼠标悬浮显示编辑图标，点击跳转编辑页面（和详情页布局一样，只是可编辑状态）
  • 更新技能：来源为市场的技能，比较本地版本与市场版本，若市场版本号更高，显示更新按钮；点击更新按钮，自动安装最新版，显示更新动画，更新成功后提示"更新成功！"
  • 风险检测：安装时异步检测技能风险，存在风险的技能高亮标记并禁止发布

2.2 Skill 市场模块

数据来源 skill.xfyun.cn，初次按技能清单获取并保存服务器。后续定期更新，更新频率：7天/次，如有新版本则更新保存，没有新版本则不变。

Skill获取接口说明

详见 skill市场接口文档.html

接口能力说明
技能列表（分页）GET /api/web/skills，支持搜索、分类、排序、分页
技能详情GET /api/web/skills/{namespace}/{slug}
版本列表GET /api/web/skills/{namespace}/{slug}/versions
技能分类GET /api/web/labels
文件列表GET /api/web/skills/{namespace}/{slug}/versions/{version}/files
单文件下载GET /api/web/skills/{namespace}/{slug}/versions/{version}/file?path={filePath}
关键词搜索GET /api/web/skills?q={q}

界面功能

  • 搜索筛选区：包含关键字搜索框
  • 分类筛选：分类包括全部、官方精选、养虾挑战赛、编程、自动化、效率、娱乐、图像、通讯、健康
  • 技能卡片：显示圆角方形图标、分类、名称、官方标签、描述（两行文字，超出显示...）、版本号、更新时间、安装量（鼠标悬浮显示中文提示）
  • 状态显示：
    • 未安装：显示安装按钮
    • 已安装已发布：显示已安装标签
  • 分页：每页20个，自适应显示，底部显示分页
  • 卡片大小：和企业skill一样大

交互功能

  • 分类筛选：点击分类标签筛选对应技能
  • 排序切换：选择排序方式重新排列列表
  • 安装技能：点击安装按钮，显示安装动画，完成后提示"安装成功"，并将技能放入企业skill模块
  • 查看详情：点击卡片跳转详情页（新页面），和企业skill模块的技能详情页一样
  • 返回列表：详情页可返回市场列表页

2.3 技能详情页

界面功能

  • 返回按钮：返回列表页/回收站
  • 状态标签：右上角显示
    • 企业skill：已发布/已下架标签
    • 市场skill：已安装标签，或安装按钮（未安装时）
  • 基础信息：技能图标、名称、描述、版本号
  • 来源信息：来源、创建者、使用次数、更新时间
  • 技能文件：文件列表，点击后右侧预览/源代码框中显示文件内容
  • 脚本模块：显示脚本文件列表（1个或多个），点击可预览
  • 参考文档：显示文档列表（1个或多个），点击可预览

交互功能

  • 保存技能：保存技能信息修改
  • 预览/源码切换：技能文件支持预览和源代码两种查看模式
  • 安装：详情页安装按钮点击后显示安装动画，完成后提示安装成功

2.4 技能新增/编辑页面

界面功能

  • 返回按钮：返回列表页
  • 基础表单：名称输入框、描述文本框、版本号输入框（文本框可修改）
  • 技能文件：文件列表，点击后右侧预览/源代码框中显示内容，可编辑
  • 脚本模块：显示/新增/上传脚本文件
  • 参考文档：显示/新增/上传参考文档
  • 文件操作：每个文件后面有删除图标，支持删除

交互功能

  • 上传文件：拖拽或点击上传脚本/文档文件
  • 新建文件：点击新建脚本/文档文件
  • 删除文件：点击文件后面的删除图标删除文件
  • 保存技能：保存新增/编辑的技能

2.5 回收站

新窗口打开显示

界面功能

  • 页面标题：显示"技能回收站"
  • 退出按钮：返回技能列表
  • 技能计数：显示共N个已删除技能
  • 技能卡片：布局和企业skill模块的技能卡片一样，图标在左侧，名称和描述在右侧，来源标签在名称后面；显示版本号

交互功能

  • 悬浮操作：鼠标悬浮显示还原和彻底删除按钮
  • 还原技能：点击后恢复至企业技能列表
  • 彻底删除：点击后永久删除技能
  • 查看详情：点击卡片查看技能详情，详情页可返回回收站

2.6 导入功能

界面功能

  • 拖拽窗口：显示文件拖动窗口，支持将zip文件拖动至窗口上传
  • 格式提示：提示".zip需要包含skill.md文件"

交互功能

  • 导入成功：拖拽文件后提示"技能导入成功！"，默认下架
  • 风险检测：同时异步检测技能风险
  • 发布提示：如果在风险监测中，点击发布提示"该技能在风险监测中，请等待检测完成后发布！"
  • 风险提示：如技能存在风险，无法发布并高亮标记：风险（鼠标悬浮显示风险原因）；点击发布提示"该技能存在风险，无法发布！"，可删除

2.7 技能详情

详情页内容

  • 内容映射：名称和描述字段，对应接口中的displayName和summary字段；
  • 文件：如下

    • 技能文件

    skill.md（有且只有一个）

    脚本文件后缀清单

    ".py", ".json", ".yaml", ".yml", ".js", ".mjs"

    参考文档后缀清单

    ".md", ".txt", ".csv", ".png", ".jpg", ".jpeg", ".svg"

三、非功能性需求

4.1 性能需求

  • 列表加载时间不超过2秒
  • 分页切换响应及时

4.2 兼容性

  • 支持主流浏览器：Chrome、Firefox、Safari、Edge
  • 响应式布局适配不同屏幕宽度

4.3 交互反馈

  • 所有操作提供明确的成功/失败提示
  • 删除等重要操作需二次确认
  • 安装等耗时操作提供加载动画
